Line Mode Browser Guide

The World-Wide Web line-mode browser allows you to find information by following references and/or by using keywords. It provides universal access to the multiverse for users on character-based terminals.

Following References

References are numbers displayed in [brackets] after particular phrases. To navigate, simply type the number and press RETURN to retrieve more information on that subject.

Searching with Keywords

Some documents act as indexes. These allow you to search for specific information using keywords. To search, use the find or f command followed by your terms.

Example: Typing f sgml examples searches the index for items containing both "SGML" and "EXAMPLE."

  • The find command is only available when it is explicitly present in the command prompt.
  • You may omit the f if your first keyword does not conflict with existing commands.
