HTML Tags Reference

<title> Tag

The <title> element defines the document title.

<title>Page Title</title>

<a> Tag

The <a> tag defines a hyperlink to another resource or fragment. Example:

<a href="../markup.html">HTML</a>

<base> Tag

The <base> tag specifies a base URL for relative links.

<base href="../../technical/addressing/syntax-addressing/naming-schemes.html">

Headings <h1>–<h6>

Use headings to structure your document. Example of a section heading:

<h2>Section Title</h2>

Other useful links include relative name, hash sign, network address format, and AAP as a placeholder.
